# Pizza Sales Analysis using SQL & Tableau

## Author
Syed Aasia Afsheen

## Project Overview
This project analyzes pizza sales data using SQL and Tableau to identify business insights such as:
- Total Revenue
- Total Orders
- Average Order Value
- Average Pizzas per Order
- Best-selling Pizzas
- Sales by Category and Size
- Peak Order Hours
- Daily and Monthly Sales Trends

## Tools Used
- MySQL
- Tableau
- GitHub

## Files
- pizza_sales.sql – SQL queries
- pizza_sales_dashboard.twb – Tableau workbook
- dashboard.png – Dashboard screenshot

## Key Insights
- Identified top-selling pizzas.
- Analyzed sales trends by day and month.
- Found peak ordering hours.
- Compared pizza categories and sizes by revenue and quantity.

## Note
This project was created for learning and portfolio purposes using a publicly available pizza sales dataset. The SQL analysis and Tableau dashboard were implemented by me.
